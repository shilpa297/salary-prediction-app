AI Salary Prediction App 💼

An AI-powered Salary Prediction Web Application built using Machine Learning, XGBoost, Natural Language Processing (TF-IDF), and Streamlit.

This project predicts the estimated annual salary of a candidate based on:

Years of Experience
Qualification
Work Type
Company Size
Job Role
Skills
Preference
🚀 Technologies Used
Python
Pandas
Scikit-learn
XGBoost
TF-IDF Vectorization
Streamlit
Machine Learning
NLP
📊 Machine Learning Models Used

The following regression models were implemented and compared:

Linear Regression
Random Forest Regressor
XGBoost Regressor

XGBoost provided the best performance with the lowest Mean Absolute Error (MAE).
